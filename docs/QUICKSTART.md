# Quick Start Guide

Get started in 5 minutes.

## Installation

Choose one:

```bash
# skills CLI (Recommended)
npx skills add SwiftyJourney/requirements-engineering-skill

# Manual
git clone https://github.com/SwiftyJourney/requirements-engineering-skill.git
cp -r requirements-engineering-skill/requirements-engineering ~/.your-agent/skills/
```

Or download `requirements-engineering.skill` from Releases and upload to Claude.ai.

## First Use

Try this:

```plaintext
Help me refine: "As a user, I want to see a feed"
```

You'll get:

- Clarifying questions
- BDD stories with multiple narratives
- Use cases with error handling
- Flowcharts and architecture diagrams
- README structure

## Common Use Cases

**Refine vague requirements:**

```plaintext
Help me refine: "Users need a dashboard"
```

**Full documentation:**

```plaintext
Create BDD stories, use cases, and diagrams for a notification system
```

**Generate diagrams:**

```plaintext
Create flowcharts for a shopping cart with offline support
```

## Tips

✅ Be specific: "Social media feed with offline caching"
✅ Mention users: "Customers and admins need different dashboards"
✅ Ask about errors: "What if there's no connectivity?"

## What You'll Get

- **BDD Stories** - Multiple narratives with scenarios
- **Use Cases** - Happy paths and error handling
- **Diagrams** - Flowcharts and architecture (Mermaid)
- **Documentation** - Structured README
